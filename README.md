# iPhonegame

Pygameで作られたブラウザ対戦シューティングゲームです。

## プレイ

**https://minango.github.io/iPhonegame/**

ブラウザ上で直接プレイできます。インストール不要です。

## ゲーム概要

画面下側の青いキャラクター（プレイヤー）を操作し、上から迫ってくる赤い敵（CPU）を倒すシューティングゲームです。

- 画面サイズ: 480 x 800（縦型）
- 操作: 画面上のボタン（L / R / SHOOT）でタッチ操作対応

### 難易度選択

| 難易度 | プレイヤーHP |
|--------|------------|
| Easy   | 20         |
| Normal | 10         |
| Hard   | 5          |
| Ultra  | 1          |

### CPレベル選択（1〜10）

レベルが上がるほど敵の移動速度・射撃頻度・追跡精度が上昇します。

| レベル | 射撃頻度 | 移動速度 | 追跡強度 |
|--------|---------|---------|---------|
| 1      | 低      | 遅      | 弱      |
| 10     | 高      | 速      | 強      |

## 技術スタック

- [Pygame](https://www.pygame.org/) — ゲームロジック・描画
- [Pygbag](https://pygame-web.github.io/) — PygameゲームをWebAssembly(WASM)に変換
- GitHub Actions — ビルド & GitHub Pages 自動デプロイ
